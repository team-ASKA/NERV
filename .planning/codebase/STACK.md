# Tech Stack

**Frontend Runtime & Frameworks**
- **Core Framework**: React 18.3 (`react`, `react-dom`)
- **Build Tool / Dev Server**: Vite (`vite`, `@vitejs/plugin-react`)
- **Language**: TypeScript (`typescript`)
- **Routing**: React Router DOM v6 (`react-router-dom`)
- **Styling**: Tailwind CSS (`tailwindcss`, `autoprefixer`), `clsx`

**Frontend Libraries**
- **Forms & Validation**: React Hook Form (`react-hook-form`)
- **Animations / 3D**: Framer Motion (`framer-motion`), Spline (`@splinetool/react-spline`, `@splinetool/runtime`)
- **Charts & Visualizations**: Chart.js (`chart.js`, `react-chartjs-2`)
- **Editor**: Monaco Editor (`@monaco-editor/react`)
- **Markdown Parsing**: `react-markdown`
- **PDF Rendering**: `pdfjs-dist`
- **Icons**: `lucide-react`, `react-icons`
- **Local Storage/IndexedDB**: `idb`, `react-use` (hooks)

**Backend Runtime & Frameworks**
- **Runtime**: Node.js
- **Framework**: Express (`express`)
- **Language**: TypeScript (`ts-node`, `typescript`)
- **Middleware**: `cors`, `dotenv`

**Cloud & Platform**
- **Hosting**: Vercel (indicated by `vercel.json` and `@vercel/node`)
- **Workspaces/Monorepo**: Setup with concurrent root and `server/` configs.
